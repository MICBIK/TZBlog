运行后端代码快速审计工作流。

使用 5 个专业审计代理并行执行：
1. 安全审计 - OWASP Top 10、认证授权
2. 性能审计 - 数据库查询、缓存策略
3. 代码质量 - 复杂度、错误处理
4. 架构审计 - 分层、依赖关系
5. 测试审计 - 覆盖率、测试质量

预计执行时间: 5-10 分钟
Token 消耗: 约 200k-400k

## 何时使用

- 完成功能开发后，提交前的质量检查
- 定期代码审计（每周/每月）
- 重大重构后的验证
- 准备发布前的最终检查

## 输出

生成详细审计报告，包含：
- 发现的问题列表（按严重程度分类）
- 各维度评分
- 修复建议
- 整体建议

## 调用方式

直接执行 `/backend-quick-audit` 或告诉 Claude：
"运行后端快速审计"

## 注意事项

- 工作流文件位于 `.claude/workflows/` 目录
- 该目录不属于前后端分支范围
- 如需修改工作流，请在独立分支或 main 分支操作
- CI 会检查分支文件范围，workflow 文件不能在 feature/backend/* 或 feature/frontend/* 分支中
