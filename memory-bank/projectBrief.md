# TZBlog 项目概述

## 项目定位
个人博客站点，以内容创作为核心，突出个人品牌的专业技术博客平台。

## 核心特点
1. **内容优先**：专注高质量技术文章、教程、经验分享
2. **个人品牌**：定制头像服务、作者信息突出展示
3. **社交互动**：关注、点赞、评论等社交功能
4. **商业化潜力**：付费内容、会员订阅、知识付费

## 技术栈
### 前端
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS v4 + shadcn/ui
- Framer Motion + TanStack Query

### 后端
- Go + Gin + GORM
- PostgreSQL + Redis
- Meilisearch (全文搜索)

## 目标用户
技术开发者、产品经理、设计师为主，技术爱好者为辅
