## 🚀 运行测试

### 基本命令

```bash
# 运行所有测试
pnpm test:e2e

# 运行测试（带 UI）
pnpm test:e2e:ui

# Debug 模式
pnpm test:e2e:debug

# Headed 模式
pnpm test:e2e:headed
```

### 按浏览器运行

```bash
pnpm test:e2e:chromium
pnpm test:e2e:firefox
pnpm test:e2e:webkit
pnpm test:e2e:mobile
```

### 查看报告

```bash
pnpm test:e2e:report
```
