## 前端部署 (Vercel)

### 优势
- ✅ 自动CI/CD，Git推送即部署
- ✅ 全球边缘网络
- ✅ 自动HTTPS
- ✅ 预览环境（PR自动部署）
- ✅ 免费额度慷慨

### 配置
```json
// vercel.json
{
  "rewrites": [
    {
      "source": "/api/:path*",
      "destination": "https://api.tzblog.com/:path*"
    }
  ],
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        {
          "key": "X-Frame-Options",
          "value": "DENY"
        }
      ]
    }
  ]
}
```

## 后端部署选择

### 方案A：腾讯云轻量应用服务器
- **配置**: 2核4G，5M带宽
- **成本**: ¥74/月
- **适合**: 初期流量小，预算有限

### 方案B：阿里云ECS
- **配置**: 2核4G，按量付费
- **成本**: ¥150-300/月
- **适合**: 流量中等，需要灵活扩展

### 方案C：Docker + K8s (后期)
- **成本**: ¥500+/月
- **适合**: 流量大，需要高可用
